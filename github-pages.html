🍎 Share Your Mac APP

一个简单而强大的Mac应用程序扫描与可视化工具，帮助您发现、统计和分享您Mac上安装的应用程序。

🚀 立即体验 📂 GitHub 仓库
🔍

智能扫描

自动扫描Mac上所有已安装的应用程序，获取详细信息包括版本、大小、安装日期等。

📊

多种视图

支持列表视图、词云视图和统计图表视图，多角度展示您的应用程序。

🎨

智能分类

自动将应用程序分类为开发工具、通讯软件、生产力工具等16个专业类别。

✨ 主要特性

🌈 美观界面

现代化的毛玻璃效果和动态背景

📱 响应式设计

完美适配不同屏幕尺寸

🎮 演示模式

内置演示数据，无需扫描即可体验

🔍 实时搜索

支持分类筛选和应用名称搜索

🚀 快速开始

  1. 克隆仓库到本地
  2. 运行 ./scan_apps.sh 扫描应用
  3. 打开 index.html 查看结果
  4. 或点击"加载演示数据"直接体验
